Frequently asked questions

Everything you need to know about MCP Hub

What is MCP (Model Context Protocol)?

MCP is an open standard by Anthropic that allows AI agents to connect to external tools and data sources. Think of it as USB for AI — plug in any tool and your agent can use it.

How is MCP Hub different from other solutions?

MCP Hub provides enterprise-grade security (OAuth, RBAC, audit logging) out of the box. Other solutions focus on individual agents — we focus on managing 10+ agents at scale.

Is the Free tier really free?

Yes. Free tier includes 1 agent, 3 MCP servers, basic OAuth, and 100 audit log entries. No credit card required.

Can I self-host MCP Hub?

Yes. MCP Hub is open source under MIT license. You can self-host on your own infrastructure. Docker support included.

What MCP servers are pre-configured?

GitHub, Slack, Notion, PostgreSQL, Pinecone, Stripe, Sentry, and AWS S3. More being added regularly. You can also add custom servers.
