Frequently asked questions

Everything you need to know

What is MCP?

MCP is an open standard by Anthropic that allows AI agents to connect to external tools and data sources. Think of it as USB for AI.

How is MCP Hub different?

Enterprise-grade security (OAuth, RBAC, audit logging) out of the box. Built for managing 10+ agents at scale.

Is Free tier really free?

Yes. 1 agent, 3 MCP servers, basic OAuth, 100 audit logs. No credit card.

Can I self-host?

Yes. Open source under MIT license. Docker support included.

Pre-configured servers?

GitHub, Slack, Notion, PostgreSQL, Pinecone, Stripe, Sentry, AWS S3.
